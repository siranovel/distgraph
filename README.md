## install ##

From rubygems:  
~~~
    [sudo] gem install num4distgraph
~~~

or from the git repository on github:  
~~~
    git clone https://github.com/siranovel/num4distgraph.git  
    cd helloci  
    jruby -S gem build num4distgraph.gemspec
    jruby -S gem install num4distgraph
~~~
